Data Acquisition

Used scrapy to extract music data from https://musicbrainz.org/. For this site we focused on song attributes including genre, title and artist.

  • musicbrainz_scrape.csv
  • html data

Data was acquired from http://labrosa.ee.columbia.edu/millionsong/
focusing on attributes artist, song_title, tempo, duration and key

  • dump

Cleaning

Empty values and non-ascii characters were removed from musicbrainz data.

  • removed 912 examples for character encoding and malformed data
  • removed 12690 examples for missing data
  • traversed 33144 examples total

Modify csv string data format from dump Data

  • specifically removed "'b characters

Transformation

Examples from both musicbrainz data and dump data were matched according to song_title and artist.

Select attributes tempo and duration were added to examples that matched from musicbrainz data to form transformed tables.

  • transformed_tables
