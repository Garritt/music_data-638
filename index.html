Data Acquisition

Used scrapy to extract music data from https://musicbrainz.org/. For this site we focused on song attributes including genre, title and artist.

  • musicbrainz_scrape.csv
  • html data

Data was acquired from http://labrosa.ee.columbia.edu/millionsong/
focusing on attributes artist, song_title, tempo, duration and key

  • dump

Cleaning

Empty values and non-ascii characters were removed from musicbrainz data.

removed 912 examples because of character encoding and malformed data
removed 12690 examples because of missing data
traversed 33144 examples total

Transformation

Examples from both musicbrainz data and dump data were matched according to song_title and artist.
Select attributes tempo and duration were added to examples that matched from musicbrainz data to form new tables.

  • transformed_tables

Exploration

run this tool on individual tables for data attribute type statistics.
AVG_TITLE_LEN, AVG_ARTIST_LEN, AVG_TEMPO, AVG_DURATION, MX_TITLE_LEN, MX_ARTIST_LEN, MX_TEMPO, MX_DURATION, GENRE_COUNT, MISSING_GENRES
